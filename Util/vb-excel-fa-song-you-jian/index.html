VB-Excel发送邮件

设置OutLook

  1. 管理员模式启动OutLook

    文件 -> 选项 -> 信任中心 -> 信任中心设置 -> 宏设置 -> 选择启用所有宏，并勾选加载项：将宏安全设置应用于已安装的加载项 -> 编程问题->选择从不向我发出可疑活动警告 -> 确定

设置Excel

  1. 新建一个excel文件，另存为xlsm，只有xlsm文件才可以执行宏

  2. 设置：文件 -> 选项 -> 信任中心 -> 信任中心设置 -> 宏设置 -> 启用所有宏 -> 勾选信任对VBA工程对象模型的访问 -> 确定

编写代码

  1. alt + F11 打开编辑器

  2. 编写代码

    
Private Sub 全自动发送邮件_Click()

    '要能正确发送并需要对Microseft Outlook进行有效配置

    On Error Resume Next

    Dim rowCount, endRowNo, endColumnNo, sFile$, sFile1$, A&, B&, lastM$, C&

    Dim objOutlook As Object

    Dim objMail As MailItem

    '取得当前工作表数据区行数列数

    endRowNo = ActiveSheet.UsedRange.Rows.Count

    endColumnNo = ActiveSheet.UsedRange.Columns.Count

    '取得当前工作表的名称，用来作为邮件主题进行发送

    'sFile1 = ActiveSheet.Name

    '创建objOutlook为Outlook应用程序对象

    Set objOutlook = CreateObject("Outlook.Application")

   '开始循环发送电子邮件

    'For rowCount = 2 To endRowNo
    rowCount = 2
    Do While rowCount < endRowNo

   '创建objMail为一个邮件对象

    Set objMail = objOutlook.CreateItem(olMailItem)

    With objMail
   
    '设置收件人地址，数据源所在列数

    .To = Cells(rowCount, 1)

    '设置抄送人地址（从通讯录表的'E-mail地址'字段中获得）
    '.CC = "969312613@qq.com;969312613@qq.com"
    ' Cells(rowCount, 0)

    '设置邮件主题,取值工作表名，

            .Subject = "主题：" & Date 'sFile1

  '设置邮件内容(从通讯录表的“内容”字段中获得)

  'align  单元格文本显示方式 left(向左)、center(居中)、right(向右)，默认是center, width-宽 height-高  border 单元格线粗细,bordercolor返回或设置对象的边框颜色

  'colSpan是一种编程语言，其属性可设置或返回表元横跨的列数

 sFile = "<tr>您好！<br>XXXXX！</tr>"

    sFile = sFile + "<table align='left' width='500' height='25' border= 1   bordercolor='#000000'> <tbody> "
    C = 1
    lastM = Cells(rowCount, 2)
    Do While rowCount < endRowNo And Cells(rowCount, 2) = lastM
    lastM = Cells(rowCount, 2)
    sFile = sFile + "<tr>  <td colspan ='8' align='center'> XXX明细 " + Str(C) + "</td> </tr> "
    C = C + 1

    B = 1

    For A = 1 To endColumnNo

    '数据表头中添加“X”后将不发送此字段

       If Application.WorksheetFunction.CountIf(Cells(1, A), "*X*") = 0 Then

       If B = 1 Then

         sFile = sFile + "<tr>  <td width='20%' height='25'> " + Cells(1, A).Text + "   </td> <td  width='30%' height='25'> " + Cells(rowCount, A).Text + "</td>"

         B = 0

       Else

        sFile = sFile + "<td width='20%' height='25'> " + Cells(1, A).Text + "   </td> <td  width='30%' height='25'> " + Cells(rowCount, A).Text + "</td> </tr>"

        B = 1

       End If

     End If

    Next
    rowCount = rowCount + 1
    Loop

   .HTMLBody = sFile

    '设置附件(从通讯录表的“附件”字段中获得)

  '  .Attachments.Add Cells(rowCount, 24).Value

    '自动发送邮件

    .Send

     End With

    '销毁objMail对象

    Set objMail = Nothing

    'Next
    
    Loop

    '销毁objOutlook对象

    Set objOutlook = Nothing

    '所有电子邮件发送完成时提示

     MsgBox Time & "XXX邮件发送成功！请持续跟进"

End Sub

  3. 功能为获取第一列的邮件地址，循环后面的列的内容并创建table表格发送，如果遇到第二列上下相同的行，合并多行一起发送

  4. 点击绿色小箭头可以直接运行发送邮件，或者点击调试，通过编译，逐语句，运行到光标处等测试代码

  1. 运行效果

  2. 如果设置完还是发不了邮件，重启下电脑，或者再看下工具->引用中是否引入outlook包（Microsoft Outlook 16.0 Object Library）16.0是版本号，也可以通过调试代码，看下Set objOutlook = CreateObject(“Outlook.Application”) 这一行的返回值objOutlook 是不是Nothing，如果是则是配置哪里没有配好，检查outlook和excel的配置。
