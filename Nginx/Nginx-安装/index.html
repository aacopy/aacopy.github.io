Nginx-安装

发表于2022-06-07|更新于2022-06-07|Nginx
|阅读量:
  • 下载稳定版本安装包，https://nginx.org/en/download.html
  • 例如：https://nginx.org/download/nginx-1.22.0.tar.gz
  • 上传nginx-1.22.0.tar.gz至/usr/local/src包下
  • tar -zxvf nginx-1.22.0.tar.gz 解压
  • cd nginx-1.22.0
  • ./configure --with-http_stub_status_module --with-http_ssl_module
  • make & make install
  • /usr/local/nginx/sbin/nginx -v
  • 启动/usr/local/nginx/sbin/nginx
