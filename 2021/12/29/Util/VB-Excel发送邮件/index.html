VB-Excel发送邮件

发表于2021-12-29|更新于2022-06-06|Util
|阅读量:

设置OutLook

  1. 管理员模式启动OutLook

    文件 -> 选项 -> 信任中心 -> 信任中心设置 -> 宏设置 -> 选择启用所有宏，并勾选加载项：将宏安全设置应用于已安装的加载项 -> 编程问题->选择从不向我发出可疑活动警告 -> 确定

设置Excel

  1. 新建一个excel文件，另存为xlsm，只有xlsm文件才可以执行宏

  2. 设置：文件 -> 选项 -> 信任中心 -> 信任中心设置 -> 宏设置 -> 启用所有宏 -> 勾选信任对VBA工程对象模型的访问 -> 确定

编写代码

  1. alt + F11 打开编辑器

  2. 编写代码

  3. 功能为获取第一列的邮件地址，循环后面的列的内容并创建table表格发送，如果遇到第二列上下相同的行，合并多行一起发送

  4. 点击绿色小箭头可以直接运行发送邮件，或者点击调试，通过编译，逐语句，运行到光标处等测试代码

  1. 运行效果

  2. 如果设置完还是发不了邮件，重启下电脑，或者再看下工具->引用中是否引入outlook包（Microsoft Outlook 16.0 Object Library）16.0是版本号，也可以通过调试代码，看下Set objOutlook = CreateObject(“Outlook.Application”) 这一行的返回值objOutlook 是不是Nothing，如果是则是配置哪里没有配好，检查outlook和excel的配置。
