RocketMQ-安装

发表于2022-02-12|更新于2022-06-06|RocketMQ
|阅读量:

版本信息

  • RocketMQ 4.9.2

  • JDK 1.8

  • CentOS 7

以下为官方推荐

  1. 64bit OS, Linux/Unix/Mac is recommended;(Windows user see guide below)
  2. 64bit JDK 1.8+;
  3. Maven 3.2.x;
  4. Git;
  5. 4g+ free disk for Broker server

下载安装包

  • 下载页面：https://rocketmq.apache.org/dowloading/releases/

  • 二进制文件下载：https://dlcdn.apache.org/rocketmq/4.9.2/rocketmq-all-4.9.2-bin-release.zip

将下载的文件上传到服务器，解压

启动服务

注意事项：

如果是非生产环境，本地测试，如果本地服务器的内容不够，会启动报错error='Cannot allocate memory',需要先调整启动参数

  • nameServer

    • 默认配置需要4G内存
    • 本地学习测试修改为256M
    • 找到JAVA_OPT="${JAVA_OPT} -server -Xms4g -Xmx4g -Xmn2g -XX:MetaspaceSize=128m -XX:MaxMetaspaceSize=320m"
    • 修改为JAVA_OPT="${JAVA_OPT} -server -Xms256m -Xmx256m -Xmn256m -XX:MetaspaceSize=128m -XX:MaxMetaspaceSize=320m"

  • broker

    • 默认配置需要8G内存
    • 本地学习测试修改为512M
    • 找到JAVA_OPT="${JAVA_OPT} -server -Xms8g -Xmx8g
    • 修改为JAVA_OPT="${JAVA_OPT} -server -Xms512m -Xmx512m"
