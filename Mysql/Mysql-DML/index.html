更新

删除

  • delele 会把删除的操作记录给记录起来，以便数据回退，不会释放空间，而且不会删除定义。
  • truncate不会记录删除操作，会把表占用的空间恢复到最初，不会删除定义
  • drop会删除整张表，释放表占用的空间
  • 删除速度：drop > truncate > delete
